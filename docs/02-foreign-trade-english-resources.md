# 外贸英语与 B2B 实务：资源清单

> **文档结构**：资源分为 **免费资源**（默认学习路径）与 **付费 / 可能付费资源**（单独成节）。两类都保留；想零成本只跟前半即可，有预算再对照后半节选用。
>
> 各网站政策会变（注册墙、证书费、课包），**以官网实时规则为准**。

> 学习原则：先懂行（Incoterms、责任与费用）再堆「900 句」；写作用**场景骨架**（10 类 × 核心句），口语用**跟读 + 复述条款**。
